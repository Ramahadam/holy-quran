# Madani Mushaf Images

This directory should contain full-color page images from the King Fahd Glorious Quran Printing Complex (Madani Mushaf).

## Required Files

- `001.png` through `604.png` (zero-padded 3-digit page numbers)
- Format: PNG or JPEG
- Recommended resolution: 1260px or 1920px width
- Source: King Fahd Complex Madani Mushaf

## How to Obtain Images

### Option 1: Download from Quran Android Repository (Recommended)

1. Visit: https://github.com/quran/quran_android/releases
2. Download one of these image databases:
   - `quran_images_1260.zip` (1260px width)
   - `quran_images_1920.zip` (1920px width - higher quality)
3. Extract the zip file
4. Copy all PNG files (page001.png through page604.png) to this directory

### Option 2: Use the Download Script

Run the provided script to attempt automatic download:
```bash
./scripts/download_sample_mushaf_images.sh
```

### Option 3: Extract from Official Apps

Download the official Quran.com or similar trusted app and extract images from the app data.
